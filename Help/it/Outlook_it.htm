Microsoft Outlook ti permette di esportare i dati in formato CSV (comma separated values), perfetto per l'importazione in Tustena.
  • Avvia Outlook.
  • Seleziona File | Import and Export....
  • Scegli Esporta in un file e clicca su Avanti.
  • Scegli Comma Separated Values (Windows) e clicca Avanti.
  • Seleziona la cartella contatti e clicca Avanti. Puoi selezionare anche più di una cartella.
  • Scegli il nome del file e clicca su Avanti.
  • Clicca Fine; il file è pronto per essere importato.
