ARCHIVE Help Page

The ARCHIVE menu is where you keep the documents which you would like to be accessible to Tustena users from any PC. It consists of the following submenus:
Categories
Here you can add, edit or delete document categories. Use this page to create the folders (categories) of documents which you want to keep in Tustena. Make new folders which can be sub-folders of any other category. 		Documents
Here you can upload documents into Tustena. You can save them under any category that you have created. You will then be able to link them to companies, contacts, activities, and opportunities.
