Exporting from GoldMine 4.0
  • Launch GoldMine 4.0.
  • Choose Export Records from the Tools menu.
  • Select Export to a new file and DBF file, and click Next.
  • In the list of GoldMine Fields on the left side of the dialog, select all of the fields, and click Add Field. Then click Next.
  • Choose the location for the export file, and click Next.
  • Select No when asked if you want to save these settings, and click Next.
  • Click Finish.
  • After the export finishes, locate the exported file and change its file extension from .dbf to .csv. The file is now ready for import.


    To export contact data from GoldMine 5.0, follow these steps:
  • Launch GoldMine 5.0.
  • Choose Tools | Import/Export Wizard | Export Contact Records.
  • Select Export to a new file and ASCII file, and click Next.
  • Select ALL Contact Records! in the drop-down list, and click Next.
  • In the list of GoldMine Fields on the left side of the dialog, select the fields you want to export, and click Add Field. We recommend you select all fields, except the notes field.
  • Click Next.
  • Choose the location for the export file, select the Export GoldMine field names... checkbox, and then click Next. Select No when asked if you want to save these settings, and click Next.
  • Click Finish.
  • After the export finishes, locate the exported file and change its file extension from .txt to .csv. The file is now ready for import.
