To export contact data from Palm Desktop, follow these steps:
  • Open the Address Book in the Palm Desktop. If you only want to export specific contacts, select those records.
  • Choose Export from the File menu.
  • In the "Export To File" dialog, enter a name for the file and choose a folder for it. In the Export as drop-down list, choose "Comma Separated (*.csv;*.txt)". Select the range of records to export - either All or Currently selected records.
  • Click Export.
  • In the "Specify Export Fields" dialog box, select the Address Book fields you want to export, and click OK. The file is now ready for import.
