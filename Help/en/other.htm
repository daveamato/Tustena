To export contact data from other programs, follow these steps:
You can import data into the system from any other application that can create a CSV (comma separated values) file.

  • Save your data source as a CSV file.
  • Ensure your file includes only one name per field.
  • Ensure your file separates first names, last names and titles into separate fields. The system cannot accept fields containing both names and titles.
  • Ensure your file includes only one phone number per field.
