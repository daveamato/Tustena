Per esportare i dati dal Palm effettua i seguenti passaggi:
  • Apri l'Address Book. Se vuoi esportare solo specifici contatti, selezionali.
  • Vai nel menu File | Export.
  • Scegli una cartella nel quale salvarlo e un nome; seleziona Comma Separated (*.csv;*.txt)". Decidi se esportare tutti i dati o solo alcuni.
  • Clicca Export.
  • Nella finestra "Specify Export Fields" seleziona i campi da esportare e clicca OK. Il file è ora pronto per essere importato in Tustena
