REPORT - Help Page

I REPORT servono per analizzare o visualizzare i dati all'interno di Tustena. Il menu report è suddiviso nei seguenti sottomenu:
Colpo d'occhio
In questa parte di tustena puoi trovare delle analisi grafiche dei tuoi dati. E' sufficente scegliere il tipo di report ed esso verrà creato in automatico. I report a "colpo d'occhio" non possono essere esportati. 		Report
Nella pagina dei report puoi creare report personalizzati che includano ogni tipo di dato in Tustena. E' sufficiente dare il nome al report e scegliere i campi da includere. I report possono risultare molto comodi da usare con la stampa unione di Word per l'invio di pubblicità o comunicazioni a mezzo posta ordinaria. I report possono essere visti in formato HTML o esportati in vari tipi di file (.csv, .xml, .pdf, .rtf, .ado recordset).
